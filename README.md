# Sprite splitter

## Requirements
Linux with `imagemagick` installed.

## Usage

```bash
$ ./sprite-splitter -i input.png -o output
```

## What is it?
This was created during my learning to make games, it is used to split the sprites of a transparent spritesheet. See example below:
